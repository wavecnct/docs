# Security & Compliance

## Compliance

### SOC 2 Type 2

Wave Connect has a SOC 2 Type 2 attestation for Security, Confidentiality, and Availability. More information is available [here](https://app.sprinto.com/trust-center/view/054381d9-6390-4e12-a7cd-57d762e9092a). To receive a copy of the report please [Contact Support](mailto:support@wavecnct.com).

### CSA Star Level 1

The Security, Trust, Assurance, and Risk (STAR) Registry is a publicly accessible registry that documents the security and privacy controls provided by popular cloud computing offerings.

Our assessment is available on the [CSA Website](https://cloudsecurityalliance.org/star/registry/wave-connect/services/wave-app)

### GDPR

We are currently working on getting GDPR compliant.

### How is my data process

Informations on how your data is processed is included in our [Privacy Policy](https://wavecnct.com/policies/privacy-policy).

### Data Encryption

Data is encrypted at rest on the underlying storage media. Communications to the Wave Connect application are encrypted using HTTPS/TLS 1.3. Certificates are issued by established third-party certificate authorities.

### Data Backup

Wave Connect backs-up customer data daily and each backup is persisted for 30 days.

<Callout type="info">These backups are not available to customers and are created for use in case of disaster.</Callout>
### Penetration testing and Audit scans

Wave Connect conducts regular penetration testing through third-party penetration testers, and has daily code reviews and static analysis checks.
